
# Carga electrónica de 6A "CE6A"


> [!NOTE]
> Docu in Galician language

Tratase dun circuíto clásico de carga electrónica, que permite paralelizar varias para obter maior corrente de carga.
Tamén podemos cambiar MOSFET un de maior corrente (VGS min<=12V\| VGS sat)
Unha opción e por un TOPFET-N que xa conta con driver interno e proteccóns.
Resumindo é unha carga electrónica mí­nima con posibilidades de maior control mediante microcontrolador...
